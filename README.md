# 🧮 Calculadora JS - Dark Theme

Uma aplicação de calculadora web funcional e responsiva, desenvolvida com foco em práticas modernas de **CSS Grid** e **Flexbox**. O projeto apresenta uma interface inspirada em temas *dark*, utilizando uma paleta de cores sóbria e efeitos de transição suaves.

---

## 🚀 Funcionalidades

* **Operações Matemáticas**: Realiza cálculos de adição, subtração, multiplicação e divisão.
* **Interface Responsiva**: Centralizada verticalmente e horizontalmente no navegador utilizando Flexbox.
* **Layout Otimizado**: Teclado organizado através de CSS Grid para garantir alinhamento preciso dos botões.
* **Design Moderno**: Estética com tons de cinza e azul metálico, incluindo efeitos de *hover* e elevação nos botões.

## 🛠️ Tecnologias Utilizadas

* **HTML5**: Estruturação semântica dos elementos da calculadora.
* **CSS3**: 
    * Uso de variáveis nativas (`:root`) para gerenciamento de cores.
    * **CSS Grid** para a malha de botões.
    * **Flexbox** para o layout principal e centralização.
* **JavaScript (ES6)**: Lógica para manipulação do DOM e processamento das expressões matemáticas.

## 📁 Estrutura do Projeto

O projeto é organizado nos seguintes arquivos:

* `index.html`: Contém a estrutura da calculadora e os gatilhos das funções de clique.
* `style.css`: Define a identidade visual, incluindo variáveis de cores e regras de layout.
* `script.js`: Gerencia a lógica de inserção de dados e a execução dos cálculos através da função `eval()`.

## 🎨 Personalização de Cores

O design utiliza as seguintes variáveis CSS para manter a consistência visual:

| Variável | Cor | Descrição |
| :--- | :--- | :--- |
| `--hk-void-bg` | `#0b0c10` | Fundo principal e do visor |
| `--hk-panel-bg` | `#151821` | Plano de fundo da calculadora |
| `--hk-accent` | `#4a5c78` | Cor base dos botões numéricos |
| `--hk-border` | `#283142` | Bordas e botões de operadores |
